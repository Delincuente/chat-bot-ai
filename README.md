# Zenith AI - Premium AI Chatbot

A high-performance, premium AI chatbot built with React, Node.js, and Google's Gemini AI.

## Features
- **Modern UI**: Dark theme with glassmorphism and smooth animations.
- **Real-time Interaction**: Instant responses from Gemini AI.
- **Chat History**: Context-aware conversations.
- **Responsive Design**: Works perfectly on desktop and mobile.

## Prerequisites
- Node.js installed.
- A Google Gemini API Key (get it from [Google AI Studio](https://aistudio.google.com/)).

## Getting Started

### 1. Backend Setup
1. Navigate to the `server` directory.
2. Create a `.env` file based on `.env.example`.
3. Add your `GEMINI_API_KEY`.
4. Run:
   ```bash
   npm install
   npm run dev
   ```

### 2. Frontend Setup
1. Navigate to the `client` directory.
2. Run:
   ```bash
   npm install
   npm run dev
   ```

## Deployment
- **Frontend**: Deploy the `client` to Vercel, Netlify, or AWS Amplify.
- **Backend**: Deploy the `server` to Railway, Render, or Heroku.
- Remember to set the environment variables on your hosting provider.

## Tech Stack
- **Frontend**: React, Vite, Framer Motion, Lucide Icons, Axios.
- **Backend**: Node.js, Express, Google Generative AI SDK.
- **Styling**: Vanilla CSS (Modern Design System).
